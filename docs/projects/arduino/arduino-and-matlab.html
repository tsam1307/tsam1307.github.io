Arduino and MATLAB®

An introductory tutorial to dim an LED connected to the UNO R3 with MATLAB® is available at Arduino Docs. The MATLAB® Support Package for Arduino hardware can be installed for reading inputs and writing outputs from the Arduino UNO R3 board either using MATLAB® console or interactively from a GUI. The MATLAB GUI code was missing in the Arduino and MATLAB® video uploaded by MathWorks® for R2015a. So, I figured out the script for MATLAB® GUI by targeting the interactive functionality showcased in that video. I will share the code here soon.

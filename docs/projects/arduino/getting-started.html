Getting Started

An anatomy of Arduino UNO R3 board and in-depth overview of its components are available at Arduino Docs. This board contains a replaceable ATmega328P microcontroller. We also need Arduino IDE to develop and upload code files (called sketches with .ino file extension) from our computer to the board. A step-by-step guide to setup the Arduino IDE for UNO R3 board is available here. One may need to install a compatible third-party device driver (CH341SER in my case) if the IDE fails to upload sketch to the board. After proper selection of the board and COM port in Tools menu, test everything is working seamlessly by uploading the sketch from File > Examples > 01.Basics > Blink menu in the Legacy IDE (1.8.10). It blinks the on-board LED using the delay() function. The on-board LED is also attached to digital pin 13, which may be used to blink an external LED. Thereafter, we learn how to blink without delay using the millis() function. Next we play with other built-in Digital and Analog examples that can be followed with components and sensors available in our electronics kit. Later, we will make a calculator using the 16x2 LCD display and 4x4 matrix keypad delivered with the UNO kit.

Reading Analog Input

Let's follow the Analog Input tutorial. The electronic components needed to make the circuit are:

  • potentiometer
  • photoresistor and 10K ohm resistor
  • 220 ohm resistor and red LED

The voltage to the analog input pin A0 is varied either by turning the knob of the potentiometer (center pin) or the resistance divider circuit created by photoresistor and 10K ohm resistor. The outer pins are connected to the 5V and ground (GND) pins on the board.

“The photoresistor value ranges from 1M ohm in darkness to 10K ohm in daylight (10 lumen) and less than 1K ohm in bright light or sunlight (>100 lumen).” — Arduino Docs

The analogRead() function is then utilized to convert analog input voltage in 0 to 5 volts range to an integer value between 0 and 1023 by 10-bit analog-to-digital convertor (ADC) inside the microcontroller. The value returned by function controls the blink rate of in-built and external LED. The external LED must be connected with 220 ohm resistor in series to draw low current from the digital pin 13.

Printing to the Serial Monitor

Next we go to the Analog Read Serial tutorial which shows us how to print any sensor value to the Serial Monitor. The Serial Monitor window can be opened from the Tools menu (or the lens icon in the green toolbar) of the Arduino IDE. First, we use the Serial.begin() function to initialize serial communication and set the data rate in bits per second (baud) in the setup routine of the sketch. Then, the Serial.println() function is used to print out the sensor value to the serial monitor in loop.

Reading Digital State

The Digital Read Serial tutorial shows how to read the state of a switch using the digitalRead() function. In the code setup, the pinMode() function sets the digital pin connected to push button as an input for reading the button state as On (1) or Off (0). This state is then printed to the serial monitor.
